xitems Plugin Sample - Omega Trick for Ext JS

ptype:t.xitems

参照：UserName.js,EMail.js

パネルなどコンテナーに設置するアイテム定義を短縮してxtype名のみで定義するプラグインです。

コンフィグオプションの"xitems"に、配列でxtypeを羅列します。

new Ext.form.FormPanel({
    title: 'Form Panel with Trick.plugins.Define',
    padding: 20,
    width: 300,
    plugins: ['t.xitems'],
    xitems: [
        'UserName',
        'EMail'
    ],
    renderTo: Ext.get('renderarea')
});

次の例は、プラグインを設置したコンテナーに直接配置されたxitemsでない場合にも動作するサンプルです。

xitemsプラグインは、ネストする複雑なレイアウトにも対応します。

new Ext.form.FormPanel({
    title: 'Form Panel deep layout with Trick.plugins.xforms',
    padding: 20,
    width: 700,
    plugins: ['t.xitems'],
    layout: 'column',
    defaultType: 'container',
    items: [{
        columnWidth: .5,
        layout: 'form',
        xitems: [
            'UserName',
            'EMail'
        ]
    },{
        width: 25,
        html: ' '
    },{
        columnWidth: .5,
        layout: 'form',
        xitems: [
            'UserName',
            'EMail'
        ]
    }],
    renderTo: Ext.get('renderarea2')
});
