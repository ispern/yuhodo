xcolumns Plugin Sample - Omega Trick for Ext JS

ptype:t.xcolumns

参照：Company.js,Branches.js

Ext.GridPanelに設定するExt.grid.ColumnModelのコンフィグオプション"columns"の定義をxtype名の配列で定義するxcolumnsを提供します。

このプラグインを利用すると、columnsのコンフィグオプションは利用できません。

new Ext.grid.GridPanel({
    title: 'Grid Panel with Trick.plugins.xcolumns',
    width: 560,
    height: 150,
    plugins: ['t.xcolumns'],
    store: {
        xtype: 'arraystore',
        fields: [
            'company',
            'branches'
        ],
        data: [
            ['Alcoa Inc', 1],
            ['Boeing Co.', 5],
            ['Hewlett-Packard Co.', 2],
            ['Wal-Mart Stores, Inc.', 9]
        ]
    },
    colModel: new Ext.grid.ColumnModel({
        xcolumns: [
            'companycolumn',
            'branchescolumn'
        ]
    }),
    renderTo: 'renderarea'
});
