xbuttons Plugin Sample - Omega Trick for Ext JS

ptype:t.xbuttons

参照：Ok.js,Cancel.js

Ext.Panel（またはその子クラス）にコンフィグオプション"xbuttons"を提供します。

buttonsで指定するExt.Buttonクラスのxtype名を配列で指定することができるとともに、プラグインを設定したパネルに"xbuttons"プロパティから、ボタンに指定した"xname"でアクセスできるようにします。

var p = new Ext.Panel({
    title: 'Panel with Trick.plugins.xbuttons',
    padding: 20,
    width: 300,
    height: 150,
    plugins: ['t.xbuttons'],
    xbuttons: [
        'ok',
        'cancel'
    ],
    renderTo: Ext.get('renderarea')
});

p.xbuttons.ok.setText('[' + p.xbuttons.ok.getText() + ']');
p.xbuttons.cancel.setText('[' + p.xbuttons.cancel.getText() + ']');
